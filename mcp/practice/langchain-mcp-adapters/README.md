3. 사용법
- 가상환경 활성화
```bash
python -m venv venv
# 윈도우라면, source venv/scripts/activate
source venv/bin/activate # 맥
pip install -r requirements.txt
```
- `.env`파일 생성 후, 본인 api_keys 정보 입력
  - `.env.example` 참고 가능.
